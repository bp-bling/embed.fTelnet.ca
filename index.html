About

My fTelnet is a web based application meant for connecting to BBSes. It uses some fairly recent features of HTML5, so let's test and ensure everything will work for you:

Feature Supported? Notes
Canvas Required! Canvas is how the output of the connection gets displayed. Without it, you can't see what the BBS is trying to show you!
WebSocket Required! WebSocket is the protocol used to talk to the remote server. Without it, you can't connect to any BBSes!
Web Storage Optional Web Storage is sort of like Cookies, but WAY better. Without it, your recent connections and dialing directory won't be saved the next time you visit this site.

If you see any Nope! labels above, that means your browser doesn't fully support the features used by My fTelnet and you should consider upgrading your browser.

Dialing Directory

This is your own personal dialing directory. Unlock the editing features for just 3 easy payments of $29.95!

Just kidding, I haven't gotten around to making the dialing directory editable yet, so you're stuck with my three default entries for now.

Until I get around to implementing this, you can use the Quick Connect menu above to connect to the server of your choice!

# Description Hostname Port Proxy? Notes Times Called

NB: The dialing directory is powered by HTML5 Web Storage, which means the entries are saved by your browser, on your computer, not on my server in an SQL database. This means if you switch computers, or even switch browsers on the same computer, you won't see the same connections on the new computer/browser.
(Once you go back to the original computer/browser, you'll see your custom directory again.)
