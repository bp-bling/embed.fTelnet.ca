NB: The dialing directory is powered by HTML5 Web Storage, which means the entries are saved by your browser, on your computer, not on my server in an SQL database. This means if you switch computers, or even switch browsers on the same computer, you won't see the same connections on the new computer/browser.
(Once you go back to the original computer/browser, you'll see your custom directory again.)
